The paths to the csv-tables in the variable section of the following POUs
have to be specified according to your local file system (current paths include placeholders).

- Initialize/FB_ReadHeatCurve
- Initialize/FB_ReadNightMode
- Initialize/FB_ReadPressTab
- Initialize/FB_ReadPT100Tab
- Initialize/FB_ReadTempResTabVR10
- Initialize/FB_ReadTempResTabVR11

NOTE: Functions do not work with csv-tables stored on network drives.
